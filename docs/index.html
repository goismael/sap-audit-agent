How it works

Four layers. One complete audit trail.

Layer 01
📡

SAP Data Collector

Pulls journal entries, change documents, and posting period data from SAP S/4HANA via supported OData APIs. Delta queries only — no bulk extraction.

→
Layer 02
🔗

Evidence Correlator

Joins SAP events with agent action logs, reasoning logs, and approval records. Detects gaps, verifies hash chains, scores completeness 0–100%.

→
Layer 03
✍️

Narrative Engine

Gemini 2.5 Flash reads each evidence package and generates a plain-language audit narrative — answering why the agent acted, not just what it did.

→
Layer 04
📊

Report Generator

Produces a scored, graded period audit readiness report — Markdown for humans, JSON for systems. Score A through D. Critical items flagged for immediate action.

Architecture

Built on the governance patterns library.

Each layer of the audit agent is a direct application of the SAP Agent Governance Patterns — P001 through P004 are not just referenced, they are the evidence sources this product reads.

Layer 01 — SAP Data Collector

OData-only SAP integration

Collects journal entries, change documents, and posting period data using only supported SAP API pathways. Delta queries via LastChangeDateTime. Identifies agent-posted documents by matching service users against the P001 registry.

OData v2 API_JOURNALENTRYITEMBASIC_SRV Delta queries P001
Layer 02 — Evidence Correlator

Four-way join with gap detection

Joins SAP events with P003 agent action logs, P003 reasoning logs, and P002 approval records via session_id. Hash chain verification. Completeness scoring. Gap classification with audit risk levels.

P002 Approvals P003 Logs Hash verification Gap detection
Layer 03 — Narrative Engine

Gemini-powered audit narratives

Uses Google Gemini 2.5 Flash via Langchain to generate plain-language audit narratives from structured evidence packages. Two prompt templates — full agent narrative and brief human-posting confirmation.

Gemini 2.5 Flash Langchain SOX-grade prompts
Layer 04 — Report Generator

Scored, graded period report

Produces a complete period audit readiness report with executive summary, statistics table, critical items section, and all narratives. Audit readiness score A–D. Saves as Markdown and JSON.

Markdown + JSON A–D grading Critical item flagging

Built on SAP Agent Governance Patterns

P001–P004 define the permission scoping, approval gates, audit logging, and failure handling that this product reads as its primary evidence sources. Start with the patterns before building the product.
